# Introduction

Welcome to the Do11y integration test site built with Mintlify.

This page exists to test documentation observability tracking across page views,
link clicks, scroll depth, search usage, and code block interactions.

## Getting started

Follow these steps to set up your project:

```bash
npm install @example/sdk
```

Then configure your application:

```javascript
import { init } from '@example/sdk';

init({
  apiKey: 'your-api-key',
  environment: 'production',
});
```

## FAQ

<AccordionGroup>
<Accordion title="How do I install the SDK?">

Run `npm install @example/sdk` in your project directory.

</Accordion>
<Accordion title="What environments are supported?">

The SDK supports Node.js 18+ and all modern browsers.

</Accordion>
</AccordionGroup>
